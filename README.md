# FakeTikTok - 短视频社交应用产品文档

## 1. 项目概述
本项目旨在开发一款基于 React Native 的短视频社交应用，对标“抖音”的核心功能。不仅支持短视频浏览，还将融合图片、表单调查、广告等多种多媒体内容形式，打造一个多元化的内容消费与社交平台。

## 2. 核心功能模块

### 2.1 内容推荐与浏览 (首页)
- **沉浸式多媒体流**：支持全屏上下滑动切换内容。
- **多媒体类型支持**：
  - **短视频**：核心内容，支持自动播放、暂停、进度拖拽。
  - **图片/图集**：横向滑动浏览高清大图。
  - **表单调查**：嵌入式交互问卷，支持用户直接参与投票或反馈。
  - **广告**：原生流广告展示，支持点击跳转。

### 2.2 社交互动
- **点赞**：双击屏幕或点击爱心图标点赞，带有动效反馈。
- **收藏**：将感兴趣的内容保存到个人收藏夹。
- **评论**：查看及发布评论，支持回复互动。
- **分享**：支持将内容分享给好友（站内私信或第三方平台分享）。

### 2.3 个人中心
- **用户信息**：头像、昵称、简介、粉丝数、关注数、获赞数展示。
- **作品管理**：展示用户发布的视频/图片内容。
- **互动记录**：
  - **喜欢**：展示用户点赞过的内容列表。
  - **收藏**：展示用户收藏的内容列表。

### 2.4 消息与分享
- **好友分享**：便捷的分享入口，支持生成分享链接或卡片。

## 3. 技术栈规划
- **核心框架**：React Native (0.76+)
- **开发语言**：TypeScript
- **路由管理**：React Navigation
- **状态管理**：Zustand 或 Redux Toolkit
- **多媒体处理**：react-native-video, react-native-fast-image
- **UI 组件库**：Restyle 或 NativeBase (待定)
- **网络请求**：Axios / React Query

## 4. 开发计划
1. **项目初始化**：搭建 RN 环境，配置基础目录结构。
2. **UI 框架搭建**：实现底部导航栏及基础页面路由。
3. **核心功能开发**：
   - 首页多媒体流（难点：高性能列表渲染）。
   - 播放器与图片预览组件封装。
4. **交互逻辑实现**：点赞、收藏状态管理与持久化。
5. **个人中心与 mock 数据对接**。
6. **测试与优化**。
